========================================================================
       MICROSOFT FOUNDATION CLASS ライブラリ : DRAW
========================================================================


AppWizard が生成した DRAW には、Microsoft Foundation Class の基本的な
使い方が示されています。アプリケーション作成のひな型としてお使いください。

このファイルには DRAW を構成している各ファイルの概要説明が含まれています。


DRAW.MAK
    このプロジェクトファイルは Visual C++ 開発環境と互換性があります。

    また、Visual C++ に用意されている NMAKE プログラム用メイクファイルとも互換
    性があります。

    MS-DOS プロンプトからデバッグ用のプログラムをビルドするには、
nmake /f DRAW.MAK CFG="Win32 Debug"
    と入力してください。

    また、リリース用のプログラムをビルドするには、
nmake /f DRAW.MAK CFG="Win32 Release"
    と入力してください。

DRAW.H
    このファイルはアプリケーションの中心となるインクルードファイルです。この
    ファイルは他のプロジェクトの固有のインクルードファイル（RESOURCE.H  も含
    みます） をインクルードし、また CDrawApp アプリケーションクラスを宣
    言します。

DRAW.CPP
    このファイルは CDrawApp アプリケーションクラスを含むアプリケーション
    の中心となるソースファイルです。

DRAW.RC
    このファイルには、サブディレクトリ RES に保存されている、アイコン、ビット
    マップ、カーソルのインクルードを含む、プログラムで使用している Microsoft
    Windows リソースがすべてを記述します。このファイルは Visual C++ 開発環境で
    直接編集することができます。

RES\DRAW.ICO
    このファイルは、アプリケーションのアイコンとして使用されるアイコンファイ
    ルです。このアイコンはリソース ファイル DRAW.RC によってインクルード
    されます。

RES\DRAW.RC2
    このファイルは Visual C++ 開発環境で編集されていないリソースを含んでいます。
    リソース エディタで編集することができないリソースはすべてこのファイルに記述
    してください。

DRAW.CLW
    このファイルは ClassWizard  が既存のクラスの編集やクラスを新規作成すると
    きに使う情報を持っています。またこのファイルは、ClassWizard によるメッセ
    ージ マップやダイアログ データ マップの編集や生成、プロトタイプ メンバ関
    数の生成に必要な情報も持っています。

/////////////////////////////////////////////////////////////////////////////

メイン フレーム ウィンドウ：

mainfrm.H, mainfrm.CPP
    これらのファイルはフレーム クラス CMainFrame を含みます。
    フレーム クラス CMainFrame は CMDIFrameWnd クラスから派生し、MDI の
    フレームを制御します。

RES\TOOLBAR.BMP
    このビットマップ ファイルはツールバーのイメージとして使います。 ツールバ
    ーやステータスバーの初期設定は CMainFrame クラスによって構築されま
    す。ツールバーのボタンコントロールを追加したい時には mainfrm.CPP
    ファイル内の配列にしたがってツールバーのビットマップを編集してください。

/////////////////////////////////////////////////////////////////////////////

AppWizard は 1 つのドキュメントタイプと 1 つのビューを作成します。

drawdoc.H, drawdoc.CPP - ドキュメント
    これらのファイルは CDrawDoc クラスを含みます。特別なドキュメント デ
    ータを付け加えたり、CDrawDoc::Serialize を使ったファイルのセーブや
    ロード機能をインプリメントしたいときはこのファイルを編集してください。

drawview.H, drawview.CPP - ビュー
    これらのファイルは CDrawView クラスを含みます。
    CDrawView オブジェクトは CDrawDoc オブジェクトをビューするために
    使います。
    
RES\drawdoc.ICO
    このファイルは CDrawDoc クラスの MDI 子ウィンドウ用アイコンとして使
    われるアイコンファイルです。このアイコンはリソースファイル DRAW.RC
    によってインクルードされます。

DRAW.REG
    このレジストリ ファイルはフレームワークの設定法を紹介するためのサンプル
    ファイルです。
    このファイルの設定を使わずにデフォルトの RegisterShellFileTypes の設定を
    使う時には、このファイルを削除してください。



/////////////////////////////////////////////////////////////////////////////
その他の標準ファイル：

STDAFX.H, STDAFX.CPP
    これらのファイルはプリコンパイル済ヘッダー ファイル DRAW.PCH やプリコ
    ンパイル済型ファイル STDAFX.OBJ を構築するために使われるファイルです。

RESOURCE.H
    このファイルは新規のリソース ID を定義する標準のヘッダー ファイルです。
